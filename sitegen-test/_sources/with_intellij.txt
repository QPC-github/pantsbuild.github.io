##############################
Using Pants with IntelliJ IDEA
##############################

This page documents how to use Pants with
`IntelliJ IDEA <http://www.jetbrains.com/idea/>`_\.
(To use IntelliJ to work on Pants *itself*, see :doc:`intellij`.)

*******************************
Pants Build Plugin for IntelliJ
*******************************

IntelliJ can ask Pants for information about your code. This gives
IntelliJ the dependency information it needs when it indexes your code.
To integrate Pants support into IntelliJ, you use a plugin.
To learn how to use it, please see
`IntelliJ-pants-plugin README <https://github.com/pantsbuild/intellij-pants-plugin/blob/master/README.md>`_.

***********************
Generating IDEA Project
***********************

If you're using an older Pants (pre-October 2014), then IntelliJ's plugin won't
do everything you need. You'll also need to use Pants to generate an IDEA
project.

To generate an IDEA project for the code in
``src/java/com/archie/path/to:target`` and
``src/java/com/archie/another:target``, use
the :ref:`idea goal <gref_goal_idea>`.
