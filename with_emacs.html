Ensime with Pants
¶

Ensime is ENhanced Scala Interaction Mode for GNU Emacs providing code navigation, type inspection, a lot of things that modern IDE should support. To install ensime please follow the instructions at https://github.com/ensime/ensime-server

To generate ensime project file for src/java/com/archie/path/to:target one just need to run this command:

$ ./pants goal ensime src/java/com/archie/path/to:target
